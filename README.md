# Nonoiam

Nonoiam is a social media platform project focused on community interaction, content sharing, and modern user experience.

## Features

- User profiles
- Content sharing
- Community interaction
- Expandable social structure
- Admin management support
- Modern web interface

## Tech Stack

- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Purpose

This project is designed as a scalable social platform idea for user interaction, sharing, and digital community building.

## Installation

1. Upload project files to your hosting/server
2. Create a MySQL database
3. Import the database schema
4. Copy `config.example.php` to `config.php`
5. Update the database credentials
6. Run the project in your browser

## Security Note

Do not upload real credentials, API keys, or private user data to GitHub.
